# 特殊符号

## `$`

语法：
```
$ 内容... $
```
```
$$ 内容... $$
```

**例1：一对\$，改变了字体**

```
普通内容：这是普通内容这是普通内容这是普通内容

一对\$包裹的内容：$内容展示内容展示内容展示内容展示$
```

普通内容：这是普通内容这是普通内容这是普通内容

一对\$包裹的内容：$内容展示内容展示内容展示内容展示$

**例2：两对\$，改变字体，换行，文本居中**

```
一对\$包裹的内容：$$内容展示内容展示内容展示内容展示$$ 这是普通内容这是普通内容这是普通内容
```

一对\$包裹的内容：$$内容展示内容展示内容展示内容展示$$ 这是普通内容这是普通内容这是普通内容
